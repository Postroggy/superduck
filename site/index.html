04Command line

Your browser,
callable as a tool.

superduck • zsh
# 1) install the CLI globally
$npm install -g superduck-cli
# 2) bootstrap native host & permissions
$superduck setup
# 3) doctor: extension / native host / socket all wired?
$superduck doctor
✓ extension reachable
✓ native host installed
✓ socket /tmp/chrome-native-host.sock
# talk to your running Chrome
$superduck context
$superduck fetch https://api.example.com
$superduck screenshot --output page.png
